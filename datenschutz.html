Datenschutzerklärung

Diese Datenschutzerklärung informiert über Art, Umfang und Zweck der Bearbeitung von Personendaten auf dieser Webseite gemäss dem Schweizer Bundesgesetz über den Datenschutz (DSG). Verantwortliche Stelle ist die im Impressum genannte Anbieterin.

Diese Webseite wird über GitHub Pages gehostet (Anbieterin: GitHub Inc., eine Tochtergesellschaft der Microsoft Corporation, USA). Beim Aufruf der Seite verarbeitet GitHub automatisch technische Informationen wie IP-Adresse, Browsertyp, aufgerufene Seite und Zeitpunkt des Zugriffs (Server-Logfiles). Diese Datenübermittlung in die USA erfolgt im Rahmen des Betriebs der Infrastruktur; GitHub verpflichtet sich vertraglich zur Einhaltung angemessener Datenschutzstandards. Weitere Informationen: GitHub Privacy Statement.

Schriftarten

Die auf dieser Webseite verwendeten Schriftarten (EB Garamond, Lato) werden lokal von unserem eigenen Server ausgeliefert. Es findet keine Verbindung zu Google-Servern statt, es werden keine Daten an Google übertragen.

Kontaktformular

Wenn du das Kontaktformular nutzt, werden die von dir eingegebenen Daten (Name, E-Mail-Adresse, Nachricht) zur Bearbeitung deiner Anfrage verwendet und per E-Mail an die Anbieterin weitergeleitet. Die technische Zustellung erfolgt über einen externen Formular-Dienst [z.B. Formspree]; dessen Datenschutzbestimmungen gelten ergänzend. Eine Weitergabe an weitere Dritte findet nicht statt.

Cookies & Tracking

Diese Webseite verwendet aktuell keine Cookies und keine Analyse- oder Trackingdienste (z.B. Google Analytics). Es findet keine Nachverfolgung des Nutzungsverhaltens statt.
